## ТЕХНИЧЕСКИЕ ДАННЫЕ

### Требования:
- WordPress 5.9+
- PHP 7.4+
- MySQL 5.7+

### Производительность:
- До 10000 товаров
- До 1000 заказов в день
- Время загрузки < 3 сек

### Безопасность:
- Nonce verification
- Capability checks
- Sanitization/Validation
- SQL injection protection
- XSS/CSRF protection

1. Прочитайте **README.md**
2. Изучите **ACTION_PLAN.md**
3. Настройте окружение
4. Начните разработку

### During разработки:
1. Используйте **FILE_TREE.md** для навигации
2. Следуйте **TECH_REQUIREMENTS.md** для написания кода
3. Консультируйтесь с **FAQ.md** при вопросах
4. Контролируйте прогресс по **PLAN.md**

### For completion:
1. Пройдите **LAUNCH_CHECKLIST.md**
2. Проверьте все пункты из **PLAN.md**
3. Убедитесь, что все работает
